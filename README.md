# 🌆 Site GTA

Repositório de um site temático inspirado no universo do jogo **GTA (Grand Theft Auto)**. O projeto foi desenvolvido com foco em HTML, CSS e JavaScript, com o objetivo de praticar e apresentar conhecimentos em desenvolvimento web.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 📁 Estrutura de Pastas

```

Site\_GTA/
├── assets/
│   └── (imagens do site)
├── styles/
│   └── style.css
├── js/
│   └── script.js
└── index.html

````

---

## 🎮 Funcionalidades

- Página temática baseada na franquia GTA
- Layout estilizado com CSS
- Design **responsivo** para diferentes tamanhos de tela
- Efeitos visuais e interações simples com JavaScript
- Uso de imagens para ambientação e estética

---

## 🛠️ Como Executar o Projeto

1. Clone o repositório:
```bash
git clone https://github.com/Joaol324j/Site_GTA.git
````

2. Acesse a pasta do projeto:

```bash
cd Site_GTA
```

3. Abra o arquivo `index.html` no navegador.
